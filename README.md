# Chat AI theme

### Install
+ Clone this repository
   ```bash
   git clone https://github.com/ShivaShirsath/Chat-AI.git
   ```
+ Open Chrome
+ Go to Extension
+ Use Developer mode
+ Select Unpacked Option
+ Navigate to location of repository
+ Open

### Uninstall
+ Navigate to Chrome extension
+ Click on ⋮ of Extension
+ Select Remove
